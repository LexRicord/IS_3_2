Линейный конгруэнтный генератор (LCG, Linear Congruential Generator) является одним из самых простых и популярных методов генерации псевдослучайных чисел. Основная идея LCG заключается в использовании линейного уравнения для генерации последовательности чисел. Давайте рассмотрим, как работает LCG.

Основные компоненты LCG
LCG определяется следующими параметрами:

Множитель (a): целое число, которое используется для умножения предыдущего значения.
Прибавка (c): целое число, которое добавляется к результату умножения.
Модуль (n): целое число, по которому вычисляется остаток, чтобы значение оставалось в определённом диапазоне.
Начальное значение (seed, x_0): начальное значение последовательности, также называемое "зерном".
Формула LCG
Следующее значение последовательности вычисляется по следующей формуле:

x_n+1=(a*x_n+c)modn
 
x_n+1 — следующее число в последовательности.
x_n — текущее число в последовательности (начинается с начального значения 
a, c и n — параметры генератора.
Пример работы LCG
Рассмотрим пример с конкретными параметрами:

Множитель (a): 421
Прибавка (c): 1663
Модуль (n): 7875
Начальное значение (x?): 5
Для первого значения последовательности:

x_1=(421*5+1663)mod7875=3768

Для второго значения последовательности:

x_2=(421*3768+1663)mod7875=1971

И так далее, для каждого последующего значения используется предыдущее значение в формуле.

Преимущества и ограничения LCG
Преимущества:

Простота: LCG легко реализовать и понимать.
Скорость: LCG работает быстро за счёт использования простых арифметических операций.
Ограничения:

Периодичность: Период генератора ограничен значением модуля n. В худшем случае, период равен n.
Качество случайности: LCG не всегда предоставляет высокое качество случайных чисел для всех приложений, особенно для криптографических целей.
Зависимость от параметров: Качество случайных чисел сильно зависит от выбора параметров a, c, и n.
